Detailed Description

Device-specific information for an EFM32 LEUART instance.

The LEUART has a similar but incompatible API to the USART/UART.

Clock source selection for LFBCLK. From the <em_cmu.h> header, e.g. cmuSelect_ULFRCO. The value zero (mapping to cmuSelect_Error) means to leave LFB in its existing configuration.
